# Hello World

A simple Hello World HTML page with an interactive JavaScript popup.

## Features

- Beautiful gradient design
- Interactive button that triggers a popup
- Smooth animations
- Click outside or close button to dismiss popup

## Files

- `index.html` - The main HTML page
- `script.js` - JavaScript for the popup functionality

## Usage

Open `index.html` in your browser and click the button to see the popup!
